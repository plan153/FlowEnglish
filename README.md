# FlowEnglish

**간단한 영작부터 시작해서, 듣고 바로 영어로 답하는 Speaking Flow 연습 앱**

### 주요 기능
- 기본 동사 중심 한→영 영작 퀴즈
- 긍정/부정/질문 변형 연습 (PNQ)
- 음성 인식으로 바로 말하기 연습
- 듣고 즉시 영어로 답하기 (Listening → Speaking)
- Spaced Repetition (SRS) 복습 시스템
- PWA 지원 (모바일 홈화면에 추가 가능)

### 사용 방법
1. 사이트 접속
2. 한국어 문장을 영어로 영작
3. 마이크 버튼으로 말하기 연습
4. 변형 버튼으로 다양한 문장 연습

### 기술 스택
- HTML + Vanilla JavaScript
- Web Speech API (음성인식 & TTS)
- Service Worker (PWA)
- GitHub Pages
