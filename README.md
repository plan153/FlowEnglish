# FlowEnglish

**초보자를 위한 단계별 Speaking Flow 앱**
슬로건: “천천히, 하나씩, 흐름을 타다”

- Level 2: 기본 Chunk & 영작
- Level 3: 시제·의문문·변형
- Level 4: Shadowing + Listen & Respond (실전)

### 주요 기능
- 기본 동사 중심 한→영 영작 퀴즈, 
- 긍정/부정/질문 변형 연습 (PNQ)
- 시제 표현과 조동사 통한 뉘앙스
- 부사구, 전치사구 통한 장소, 방향, 시간 확장하기
- 자연스러운 발음
- 음성 인식으로 바로 말하기 연습
- 듣고 즉시 영어로 답하기 (Listening → Speaking)
- 듣고 즉시 쉐도잉 하기 (녹음 후 발음비교)
- Spaced Repetition (SRS) 복습 시스템
- PWA 지원 (모바일 홈화면에 추가 가능)
- 아이폰, 안드로이드 모두 지원하도록 설계
- (DB업데이트 쉽게 하도록 설계)

### 기술 스택
- HTML + Vanilla JavaScript
- Web Speech API (음성인식 & TTS)
- Service Worker (PWA)
- GitHub Pages

Level 1: Basic Chunks & Sentence Building
레벨 1: 기본적인 단어와 문장 구성 방법주제: 영어의 기본 뼈대 잡기 (어순 + 의미 덩어리)세부 목표영어 어순(S + V + O)을 몸으로 익히기 (한국어와 가장 다른 부분)
Chunk(의미 단위) 학습: 단어를 따로 외우지 않고 “[I drink coffee]”, “[go to school]”처럼 덩어리로 익히기
be동사(am/is/are)와 일반동사(go, eat, like)의 기본 형태 익히기
간단한 긍정문 100% 정확하게 만들 수 있게 되기
기본 발음과 리듬 느끼기

기대 학습 성과 (끝났을 때)“나는 매일 커피를 마신다” → “I drink coffee every day.” 자연스럽게 말할 수 있음
150~200개의 기본 Chunk를 알고 있음
영작 정확도 80% 이상

앱 내 주요 활동간단 영작
Chunk 카드 학습
긍정  부정 변형 (기본)
기본 Shadowing (짧은 문장 따라 말하기)

Level 2: Tenses & Questions
레벨 2: 시제와 질문하기주제: 문장을 다양하게 확장하고 질문/부정까지 자유롭게 하기세부 목표현재, 과거, 미래 시제 구분하고 사용하기
의문문 만들기 (Yes/No 질문 + Wh- 질문)
부정문 완벽하게 만들기 (don't, doesn't, didn't)
주어에 따른 동사 변화 (I go vs He goes) 마스터
같은 내용을 여러 방식으로 표현할 수 있는 능력 키우기

기대 학습 성과 (끝났을 때)“어제 뭐 했어?” → “What did you do yesterday?” 자연스럽게 만들기
“I didn’t go to school yesterday because I was sick.” 같은 문장 만들기
기본적인 일상 대화 문장을 스스로 구성 가능

앱 내 주요 활동PNQ 변형 (긍정 → 부정 → 질문)
시제 변환 연습 (Present / Past / Future)
Wh- 질문 영작
중간 길이 Shadowing (6~10단어 문장)

Level 3: Shadowing & Real Flow
레벨 3: 섀도잉 및 실제 상황에서의 연습주제: 듣고 → 이해하고 → 바로 말하기 (실전 Speaking 준비)세부 목표Shadowing을 통해 자연스러운 억양, 속도, 리듬 익히기
영어 질문을 듣고 즉시 영어로 답변하는 반응 속도 키우기
조금 더 긴 문장(10~15단어)도 자연스럽게 말하기
실생활 상황(인사, 식사, 길묻기 등)에서 바로 나오는 표현 익히기
듣기 이해도 + 발화 유창성 동시에 향상

기대 학습 성과 (끝났을 때)원어민이 천천히 말하면 80% 이상 이해하고 답할 수 있음
“How was your weekend?” → 바로 “It was good. I went hiking.” 같은 답변 나옴
말하기 자신감이 크게 상승하고, 실제 외국인과 간단한 대화 가능

앱 내 주요 활동집중 Shadowing 모드 (3회 반복 듣기 → 따라 말하기)
Listen & Respond (질문 듣고 바로 답하기)
상황별 대화 연습
녹음 비교 & 발음 피드백
Mixed Flow 연습 (Chunk + 시제 + 질문 섞음)

레벨 전체 설계 철학Level 1: 기초 탄탄히 (틀을 제대로 잡기)
Level 2: 문장 확장 (도구 늘리기)
Level 3: 실전 적용 (실제로 사용하기)

각 레벨을 끝낼 때마다 **“아, 이제 좀 할 수 있게 됐다”**는 성취감을 느끼도록 설계했습니다.
