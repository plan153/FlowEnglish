✨ 앱 특징

  • ✅ 3단계 레벨 시스템
  • ✅ 누르고 말하기 (Push-to-Talk)
  • ✅ 실시간 음성 인식 + 정답 비교
  • ✅ 따라하기(Shadowing) + 영어듣기
  • ✅ 하루 10문장 목표

📱 사용 방법

  1. 레벨 선택 후 문장 확인
  2. 🎤 말해보기 버튼을 누른 채 말하기
  3. 손을 떼면 자동 종료
  4. 📊 비교하기로 내 말 vs 정답 확인

🔊 TTS 고급 설정

iPhone: 설정 → 접근성 → Spoken Content → Voices → Samantha 선택
Android: 설정 → Google TTS → English (US) → Google US English
추천 속도: Level 1~2는 0.9, Level 3는 1.0~1.1
