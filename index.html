✨ 앱 특징

  • ✅ 3단계 레벨 시스템
  • ✅ 누르고 말하기 방식
  • ✅ 실시간 음성 인식 + 정답 비교
  • ✅ 따라하기 + 영어듣기
  • ✅ 하루 10문장 목표

📱 사용 방법

  1. 레벨 선택 후 문장 확인
  2. 🎤 말해보기 버튼을 누른 채 말하기
  3. 손을 떼면 자동으로 녹음 종료
  4. 📊 비교하기 버튼으로 내 말과 정답 비교

🔊 TTS (영어듣기) 고급 설정 방법

📱 iPhone / iPad

설정 → 접근성 → Spoken Content → Voices
English (US) → Samantha 또는 Karen 선택

🤖 Android

설정 → Google TTS 또는 음성 → English (United States)
Google US English 선택

🌐 Chrome 브라우저 (PC / 모바일)

1. 주소창에 chrome://settings/voice 입력 후 Enter
2. English (United States)에서 Google US English 또는 Microsoft David 선택
3. 속도(Speed)는 0.9 ~ 1.1 추천

🌐 Edge 브라우저 (PC / 모바일)

1. 주소창에 edge://settings/voice 입력 후 Enter
2. English (United States)에서 원하는 목소리 선택
3. Chrome과 동일하게 속도 조절 가능

추천 설정
• Level 1, 2 → 속도 0.85 ~ 0.95
• Level 3 (실전) → 속도 1.0 ~ 1.1

📲 홈 화면에 추가하기

iPhone: Safari 공유 → “홈 화면에 추가”
Android: Chrome ⋮ → “홈 화면에 추가”
